---
title: "Reports"
description: "Sales, payment, shift, inventory, and KPI reports for the point of sale."
---

**Base path:** `/api/v1/pos/reports`

**Authentication required:** `TenantOrPlatform` + `PosReportViewer`. A tenant must be selected for most endpoints.

## Common query parameters

Most report endpoints share these date range parameters:

| Parameter | Type | Description |
|---|---|---|
| `dateFrom` | `DateOnly` (YYYY-MM-DD) | Start of the report period (inclusive). |
| `dateTo` | `DateOnly` (YYYY-MM-DD) | End of the report period (inclusive). |
| `storeId` | `uuid` (optional) | Filter to a specific store. |

---

## Daily summary

```
GET /api/v1/pos/reports/daily-summary
```

Returns aggregate totals for a single business day.

### Query parameters

<ParamField query="date" type="string (YYYY-MM-DD)" required>
  The business date to summarize.
</ParamField>

### Response

<ResponseField name="date" type="string (YYYY-MM-DD)">The business date.</ResponseField>
<ResponseField name="totalTickets" type="integer">Number of completed sales.</ResponseField>
<ResponseField name="totalAmount" type="decimal">Total revenue.</ResponseField>
<ResponseField name="totalItems" type="integer">Total units sold.</ResponseField>
<ResponseField name="avgTicket" type="decimal">Average ticket value.</ResponseField>

### Example

```bash curl
curl "https://api.cobranzadigital.com/api/v1/pos/reports/daily-summary?date=2026-03-17" \
  -H "Authorization: Bearer <accessToken>" \
  -H "X-Tenant-Id: <tenantId>"
```
